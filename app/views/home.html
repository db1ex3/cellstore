What is SECXBRL.info?

A database of US public company financial information submitted to the US Securities and Exchange Commission using the global standard XBRL technical syntax

  • Developed by
  • using
  • BizQL.io

Sophisticated Semantic Processing

Semantically extract or compute any kind of information like facts across filings or fact tables for a particular component. The platform deals with amendments, allows filtering by explicit dimensions or custom mapping when retrieving facts by concepts.

Custom Components

Aggregate information from multiple report components and come up with a uniform representation by mapping or imputing missing values. The platform allows you to create your custom components that are populated in real-time allowing you to compare filings.

Performance

Queries that took hours or days to compute before, can be executed within seconds. The platform exploits latest database technology and distributes the processing across machines in the cloud.

Application Profiles

Analyze information from any kind of profile (e.g. Edgar or FINREP). The platform was designed to store and query data in any profile in real time.

  • UI

    A powerful user interface allows you to visually explore and compare data within or across entities.

  • REST API

    An easy to use REST-based API allows you to manually or programmatically extract information in many data formats to be used in Excel or for building new applications.

  • Custom Development using BizQL

    An extremely powerful and highly optimized query language (BizQL) allows you to develop your own algorithms and extend the provided REST API with only basic programming skills.
